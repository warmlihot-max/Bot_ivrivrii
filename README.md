# 🤖 Bot Telegram - Restaurant IVRIVRII

Ce projet est un bot Telegram interactif conçu pour le restaurant **IVRIVRII**. Il permet aux clients de consulter le menu, de voir les horaires, d'obtenir l'adresse du restaurant via Google Maps et de passer des commandes directement sur Telegram.

## 🛠️ Installation

Pour faire tourner ce bot en local (par exemple sur Pydroid 3 ou votre ordinateur), vous devez installer la bibliothèque requise :

```bash
pip install pyTelegramBotAPI
